# 数据分析功能更新摘要

更新日期: 2025-11-24

## 新增功能总览

本次更新为职位推荐平台的数据分析模块新增了**4大核心功能**和**3个新图表**，显著增强了数据分析能力。

---

## 1. 日期筛选功能

### 功能描述
允许用户按时间范围筛选和查看数据统计。

### 使用方法
- 在数据分析页面顶部找到「数据筛选」区域
- 选择开始日期和结束日期
- 点击「应用筛选」按钮
- 所有图表将实时更新为筛选后的数据
- 点击「重置」按钮可恢复查看全部数据

### 技术实现
- **后端**: 在statistics API中支持`start_date`和`end_date`查询参数
- **前端**: 添加日期选择器组件，自动更新所有图表

---

## 2. 技能需求分析

### 功能描述
统计和展示市场上最热门的技能需求，帮助求职者了解应该学习哪些技能。

### 数据展示
- **Top 20技能排行**: 横向柱状图清晰展示
- **需求数量**: 每个技能在职位中出现的次数
- **数据洞察**: 自动显示最热门技能及其需求数

### 技术实现
- **后端**: 遍历所有职位的tags字段，使用Counter统计技能频次
- **前端**: 使用Recharts的横向柱状图展示Top 20技能

---

## 3. 公司维度分析

### 3.1 公司类型分布
分析不同类型公司（民营、国企、外企、上市公司等）的招聘情况。

### 3.2 公司规模分布
展示不同规模公司（20人以下到10000人以上）的职位数量和薪资水平。

### 数据价值
- 了解各类型公司的招聘活跃度
- 对比不同规模公司的薪资水平
- 辅助求职者选择目标公司类型

### 技术实现
- **后端**: 通过company外键关联查询company_type和company_size
- **前端**: 使用双轴柱状图同时展示职位数和平均薪资

---

## 4. 数据导出功能

### 功能描述
一键导出统计数据为CSV文件，方便进一步分析和报告。

### 导出内容
- 城市分布数据（城市、职位数、平均薪资）
- 学历分布数据（学历、职位数、平均薪资）
- 经验分布数据（经验、职位数、平均薪资）

### 使用方法
1. 点击页面右上角的「导出数据」按钮
2. 自动下载CSV文件（文件名包含导出日期）
3. 可用Excel或其他表格软件打开

### 技术实现
- 使用Blob API生成CSV文件
- 添加UTF-8 BOM确保中文正确显示
- 客户端下载，无需服务器处理

## API更新

### 端点
```
GET /api/jobs/jobs/statistics/
```

### 新增查询参数
- `start_date`: 开始日期（格式：YYYY-MM-DD）
- `end_date`: 结束日期（格式：YYYY-MM-DD）

### 新增响应字段
```json
{
  "skills_distribution": [
    {"skill": "Python", "count": 150}
  ],
  "company_type_distribution": [
    {"company__company_type": "民营", "count": 200, "avg_salary": 12000}
  ],
  "company_size_distribution": [
    {"company__company_size": "100-499人", "count": 100, "avg_salary": 11000}
  ]
}
```

## 使用示例

### 示例1: 查看2024年的技能需求趋势
1. 访问数据分析页面
2. 设置开始日期：2024-01-01
3. 设置结束日期：2024-12-31
4. 点击「应用筛选」
5. 查看「技能需求 Top 20」图表
6. 点击「导出数据」保存结果

### 示例2: 对比不同公司类型的薪资
1. 查看「公司类型分布」图表
2. 观察不同公司类型的职位数（绿色柱）和平均薪资（橙色柱）
3. 识别薪资最高的公司类型

### 示例3: API直接访问
```bash
# 获取2024年Q1的统计数据
curl "http://localhost:8000/api/jobs/jobs/statistics/?start_date=2024-01-01&end_date=2024-03-31"
```

## 数据价值

### 对求职者
- **技能提升方向**: 通过技能需求排行了解应该学习哪些技能
- **公司选择**: 对比不同类型和规模公司的薪资水平
- **时间趋势**: 通过日期筛选了解市场变化趋势
- **数据导出**: 可以导出数据做个人职业规划分析

### 对企业HR
- **薪资定价参考**: 了解同类型同规模公司的薪资水平
- **技能需求**: 把握市场热门技能，优化招聘JD
- **竞争分析**: 对比不同时间段的招聘数据
- **报告制作**: 导出数据制作招聘分析报告

### 对研究人员
- **市场分析**: 多维度分析就业市场结构
- **趋势研究**: 时间序列数据分析
- **技能演变**: 追踪技能需求变化
- **数据支持**: 导出数据用于论文或报告

## 性能优化

1. **数据库查询优化**: 使用Django ORM的聚合函数，减少数据库查询次数
2. **前端缓存**: 筛选条件变化时才重新请求数据
3. **客户端导出**: CSV导出在浏览器端完成，不增加服务器负担
4. **响应式设计**: 所有图表自适应屏幕尺寸
