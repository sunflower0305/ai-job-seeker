# 猎聘网爬虫使用指南

## 简介

本爬虫基于Playwright实现，可以**不登录**爬取猎聘网的职位数据。

## 功能特点

- ✅ **无需登录**：可以直接爬取公开职位信息
- ✅ **多城市支持**：支持北京、上海、深圳、杭州等主要城市
- ✅ **多关键词**：支持批量爬取多个职位关键词
- ✅ **数据去重**：自动去除重复职位
- ✅ **数据结构化**：自动解析薪资、公司等信息
- ✅ **无头/有头模式**：支持选择是否显示浏览器界面

## 安装依赖

```bash
pip install playwright
playwright install chromium
```

## 快速开始

### 方式1：使用运行脚本（推荐）

```bash
# 使用默认配置（北京、上海、深圳、杭州，Python、Java、Web前端、数据分析）
python run_liepin_crawler.py --headless

# 自定义城市和关键词
python run_liepin_crawler.py --cities 北京 上海 --keywords Python Java --pages 3 --headless

# 查看帮助
python run_liepin_crawler.py --help
```

### 方式2：直接使用爬虫模块

```python
from crawler.liepin_spider import LiepinSpider

# 创建爬虫实例
spider = LiepinSpider(headless=True)

# 批量爬取
spider.crawl_all(
    cities=[('010', '北京'), ('020', '上海')],
    keywords=['Python', 'Java'],
    max_pages=3
)

# 保存数据
if spider.jobs_data:
    spider.save_to_json()
```

## 命令行参数

| 参数 | 说明 | 默认值 |
|------|------|--------|
| `--cities` | 城市列表 | 北京 上海 深圳 杭州 |
| `--keywords` | 职位关键词 | Python Java Web前端 数据分析 |
| `--pages` | 每个关键词爬取页数 | 3 |
| `--headless` | 使用无头模式 | False |

## 注意事项

1. **爬取频率**：建议控制爬取速度，脚本已内置延时
2. **反爬措施**：如遇到反爬，可增加页面延时或减少页数
3. **数据用途**：仅用于学习研究，不得用于商业用途
4. **浏览器**：首次使用需要安装Chromium浏览器

## 常见问题

### Q1: 提示找不到浏览器

**A:** 运行 `playwright install chromium` 安装浏览器

### Q2: 爬取速度慢

**A:** 使用 `--headless` 参数启用无头模式，可提高速度

### Q3: 获取数据为空

**A:** 可能触发了反爬，尝试减少页数或增加延时

### Q4: 如何查看爬取过程

**A:** 不使用 `--headless` 参数，可以看到浏览器界面
