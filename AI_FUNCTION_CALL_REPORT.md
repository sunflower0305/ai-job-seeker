# AI助手功能检查报告（已更新 - 包含网络搜索）

## Function Call 使用情况

### 1. JobRecommender（简历分析推荐器）

**使用的 Function Calls: 1个**

#### `search_jobs`
- **功能**: 搜索匹配的职位
- **参数**:
  - `skills`: 技能列表
  - `position_title`: 职位名称关键词
  - `min_salary`: 最低薪资
  - `max_salary`: 最高薪资
  - `education`: 学历要求
  - `experience`: 经验要求
  - `limit`: 返回结果数量（默认10）
- **数据源**: ✅ **数据库** - 使用 `Job.objects.filter(is_active=True)`
- **返回**: JSON格式的职位列表

---

### 2. ConversationalJobAssistant（对话式AI助手）

**使用的 Function Calls: 6个**

#### 【数据库查询工具】

##### `search_jobs_by_keywords`
- **功能**: 根据关键词快速搜索职位
- **参数**:
  - `keywords`: 搜索关键词（可以是职位名称、技能等）
  - `limit`: 返回结果数量（默认5）
- **数据源**: ✅ **数据库** - 搜索职位标题和技能标签
- **查询逻辑**: `title__icontains` OR `tags__icontains`

##### `search_jobs_advanced`
- **功能**: 高级多条件搜索
- **参数**:
  - `skills`: 技能列表
  - `position_title`: 职位名称关键词
  - `min_salary`: 最低薪资
  - `max_salary`: 最高薪资
  - `city`: 城市
  - `education`: 学历要求
  - `experience`: 经验要求
  - `limit`: 返回结果数量（默认5）
- **数据源**: ✅ **数据库** - 支持多条件组合查询
- **特点**: 支持更精细的筛选条件

##### `get_job_details`
- **功能**: 获取职位详细信息
- **参数**:
  - `job_id`: 职位ID
- **数据源**: ✅ **数据库** - 通过ID精确查询
- **返回**: 包含公司信息、职位描述、要求、福利等完整信息

#### 【网络搜索工具】✨ **新增**

##### `search_web_for_jobs`
- **功能**: 在网上搜索最新的招聘信息
- **参数**:
  - `query`: 搜索查询（例如"Python开发工程师 北京 2024"）
  - `max_results`: 返回结果数量（默认5）
- **数据源**: ✅ **网络搜索** - 使用 DuckDuckGo 搜索引擎
- **特点**:
  - 自动添加"招聘 最新"关键词优化搜索
  - 内置重试机制（最多3次）
  - 随机延迟避免速率限制
  - 友好的错误提示
- **返回**: 包含标题、摘要、URL的搜索结果

##### `search_salary_trends`
- **功能**: 搜索行业薪资趋势和市场行情
- **参数**:
  - `position`: 职位名称（例如"Python开发工程师"）
  - `city`: 城市名称（可选，例如"北京"）
- **数据源**: ✅ **网络搜索** - 使用 DuckDuckGo 搜索引擎
- **特点**:
  - 自动构建薪资相关查询
  - 支持城市级别的薪资查询
  - 获取2024年最新数据
  - 内置重试和错误处理
- **返回**: 薪资行情相关的网页摘要

##### `search_company_info`
- **功能**: 搜索公司背景资料和评价
- **参数**:
  - `company_name`: 公司名称
- **数据源**: ✅ **网络搜索** - 使用 DuckDuckGo 搜索引擎
- **特点**:
  - 搜索公司评价和背景信息
  - 帮助用户了解潜在雇主
  - 内置重试和错误处理
- **返回**: 公司相关的评价和背景资料

## 数据源分析

### ✅ 使用数据库职位数据
- **3个工具连接到Django数据库**
- 使用 `Job.objects` 进行ORM查询
- 数据来自 `jobs_job` 表和关联的 `jobs_company` 表
- 查询示例:
  ```python
  queryset = Job.objects.filter(is_active=True)
  queryset = queryset.filter(title__icontains=position_title)
  jobs = queryset.select_related('company')[:limit]
  ```

  - 友好的错误提示
  - 智能查询优化

- **错误处理**:

  - 友好的错误提示

3. **错误处理**: 区分速率限制和其他错误
4. **降级策略**: 搜索失败时引导使用数据库功能

## 使用建议

### 最佳实践
1. **优先使用数据库工具**（速度快、稳定）
2. **网络搜索作为补充**（获取最新信息、行情数据）
3. **避免频繁调用**网络搜索工具（有速率限制）
4. **合理组合**数据库和网络数据

### 典型使用场景

#### 场景1：推荐职位
```
用户: "帮我推荐Python开发的工作"
AI执行:
1. search_jobs_by_keywords("Python")  # 数据库搜索
2. search_web_for_jobs("Python开发工程师 2024")  # 网络补充
```

#### 场景2：薪资咨询
```
用户: "北京Python工程师工资多少？"
AI执行:
1. search_jobs_advanced(position_title="Python", city="北京")  # 数据库数据
2. search_salary_trends("Python开发工程师", "北京")  # 网络行情
```

#### 场景3：公司调查
```
用户: "阿里巴巴这个公司怎么样？"
AI执行:
1. search_jobs_by_keywords("阿里巴巴")  # 查看数据库中的职位
2. search_company_info("阿里巴巴")  # 网络搜索评价
```
