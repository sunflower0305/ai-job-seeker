# 数据分析与可视化功能指南

## 🎉 新增功能

已成功为职位推荐平台添加完整的数据分析与可视化功能！

## ✨ 功能亮点

### 1. 后端统计API
**端点**: `GET /api/jobs/jobs/statistics/`

提供全面的职位数据统计：
- **基础统计**: 职位总数、公司总数
- **薪资统计**: 平均薪资、最低薪资、最高薪资
- **薪资区间分布**: 按薪资范围统计职位数量
- **城市分布**: Top 10城市职位数量和平均薪资
- **学历分布**: 各学历要求的职位数量和平均薪资
- **经验分布**: 各经验要求的职位数量和平均薪资
- **行业分布**: Top 10行业职位数量

### 2. 数据分析页面
**路径**: `http://localhost:3000/analytics`

完整的数据可视化页面，包含：

#### 顶部统计卡片
- 📊 职位总数
- 🏢 公司总数
- 💰 平均薪资
- 📈 薪资范围

#### 交互式图表
1. **薪资区间分布** - 柱状图
   - 展示不同薪资区间的职位数量
   - 清晰了解薪资分布情况

2. **城市职位分布** - 横向柱状图
   - Top 10城市职位数量对比
   - 快速定位热门求职城市

3. **学历要求与薪资** - 双轴柱状图
   - 左轴: 职位数量
   - 右轴: 平均薪资
   - 对比不同学历的就业机会和薪资水平

4. **工作经验与薪资关系** - 折线图
   - 蓝线: 职位数量趋势
   - 红线: 平均薪资趋势
   - 了解经验与薪资的关系

5. **行业分布** - 饼图
   - Top 10行业占比
   - 一目了然各行业招聘热度

6. **各城市平均薪资对比** - 柱状图
   - 横向对比各城市薪资水平
   - 辅助做出求职地点决策

#### 数据洞察
自动分析并展示：
- 最热门城市
- 薪资最高城市
- 主要招聘行业

### 3. 首页数据概览
**路径**: `http://localhost:3000/`

首页实时显示：
- 当前职位总数
- 合作企业数量
- 市场平均薪资
- 最热门城市

数据从后端API实时获取，动态更新！

### 4. 导航栏更新
在顶部导航栏新增「数据分析」入口，方便用户快速访问数据分析页面。

## 🚀 使用方法

## 📈 数据分析能力

### 可以回答的问题

1. **市场概况**
   - 当前有多少个职位？
   - 有多少家公司在招聘？

   - 市场平均薪资是多少？

   - 哪个城市的职位最多？

   - 各城市薪资水平对比？

4. **需求分析**
   - 哪个学历要求最常见？
   - 哪个经验要求最常见？
   - 哪个行业招聘最活跃？

5. **趋势洞察**
   - 经验越多薪资越高吗？
   - 学历对薪资影响多大？

## 📝 API文档

### 获取统计数据

**请求**:
```
GET /api/jobs/jobs/statistics/
```

**查询参数**:
- `start_date` (可选): 开始日期，格式 YYYY-MM-DD
- `end_date` (可选): 结束日期，格式 YYYY-MM-DD

**示例**:
```bash
# 获取所有数据
GET /api/jobs/jobs/statistics/

# 获取特定时间范围的数据
GET /api/jobs/jobs/statistics/?start_date=2024-01-01&end_date=2024-12-31
```

**响应**:
