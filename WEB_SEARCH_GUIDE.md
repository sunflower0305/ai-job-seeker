# 网络搜索功能使用指南

## 🎉 新功能概览

AI助手现已集成**网络搜索功能**，可以从互联网获取最新的招聘信息、行业薪资趋势和公司背景资料！

### 新增的3个搜索工具

1. **search_web_for_jobs** - 搜索最新招聘信息
2. **search_salary_trends** - 查询薪资行情
3. **search_company_info** - 调查公司背景

---

## 🚀 快速开始

### 1. 安装依赖

```bash
pip install -r requirements_ai.txt
```

新增依赖包：
- `duckduckgo-search==6.3.5` - DuckDuckGo搜索引擎
- `beautifulsoup4==4.12.3` - HTML解析
- `requests==2.31.0` - HTTP请求

### 2. 启动服务

```bash
# 启动后端
python manage.py runserver

# 启动前端（新终端）
cd frontend
npm run dev
```

### 3. 使用功能

访问 http://localhost:3000/ai-recommend 并与AI对话！

## 💬 使用示例

### 示例1：搜索最新招聘

**你说：**
> "帮我在网上搜索一下北京有哪些Python开发的职位"

**AI会：**
1. 使用 `search_web_for_jobs` 在网上搜索

### 示例2：查询薪资行情

### 示例3：调查公司背景

**你说：**
> "字节跳动这个公司怎么样？"

**AI会：**
1. 使用 `search_company_info` 搜索公司评价
2. 使用 `search_jobs_by_keywords` 查看职位
3. 综合信息帮你分析

## 🔧 技术细节

### 搜索引擎
- **使用**: DuckDuckGo
- **优点**:
  - 免费，无需API Key
  - 隐私保护好
  - 搜索质量高
- **限制**: 有速率限制（已优化处理）

### 优化特性

#### 1. 自动重试
```python
最多重试3次
每次间隔2-4秒随机延迟
```

#### 2. 智能延迟
```python
初始延迟: 1-2秒
重试延迟: 2-4秒
避免触发速率限制
```

#### 3. 友好错误提示
```
遇到速率限制 → "网络搜索暂时不可用，建议使用数据库搜索"
搜索失败 → 显示具体错误信息
无结果 → 提示调整搜索关键词
```

## ⚠️ 注意事项

### 速率限制
DuckDuckGo对搜索频率有限制：
- ✅ **正常使用**：无问题
- ⚠️ **频繁搜索**：可能触发限制
- 💡 **解决方案**：等待几分钟后重试

### 错误处理
如果看到"速率限制"错误：
1. 稍等几分钟再试
2. 使用数据库搜索功能
3. AI会自动降级到数据库查询

### 网络依赖
网络搜索需要：
- ✅ 稳定的网络连接
- ✅ 能访问DuckDuckGo
- ⚠️ 国内可能较慢
